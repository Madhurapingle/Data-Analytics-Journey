### AGGREGATE FUNCTIONS:

## 1.COUNT()

## PURPOSE:
IT RETURNS TOTAL NUMBER OF ROWS

## SYNTAX:
SELECT COUNT(*)
FROM TABLE_NAME;

## EX:
SELECT COUNT(*) 
FROM EMPLOYEES
WHERE DEPARTMENT = 'IT';

# REAL WORLD USE:
NUMBER OF EMPLOYEES WORKING IN A COMPANY

## 2.SUM()-

## PURPOSE:
IT ADDS ALL THE NUMERIC VALUE IN COLUMN 

## SYNTAX:
SELECT SUM()
FROM TABLE_NAME;

# EX-
```sql
SELECT SUM(SALARY)
FROM EMPLOYEES;
```

# REAL WORLD USE:
TOTAL AMOUNT OF SALARY A COMPANY IS PAYING TO ALL EMPLOYEES 



## 3.AVG()-

## PURPOSE:
Returns the average value of a numeric column.

## SYNTAX:
SELECT AVG()
FROM TABLE_NAME;


# EX-
```sql
SELECT AVG(SALARY) FROM EMPLOYEES;
```

# REAL WORLS USE:
WHAT IS THE AVERAGE SALARY A COMPANY IS GIVING TO ITS EMPLOYEES


## 4.MIN()-

## PURPOSE:
IT GIVES THE MINIMUM VALUE FROM ALL THE VALUES PRESENTS

## SYNTAX:
SELECT MIN()
FROM TABLE_NAME;

# EX-
```sql
SELECT MIN(SALARY) FROM EMPLOYEES;
```

# REAL WORLD USE:
WHAT IS THE MINIMUM SALARY AN EMPLOYEE IS GETTING 

## 5.MAX()-

## PURPOSE:
IT GIVES MAXIMUM VALUE FROM ALL THE VALUES PRESENT 

## SYNTAX:
SELECT MAX()
FROM TABLE_NAME;

# EX-
```sql
SELECT MAX(SALARY) FROM EMPOLYEES;
```

# REAL WORLD USE: 
WHAT IS THE MAXIMIMUM OR HIGHEST SALARY COMPANY IS PAYING
